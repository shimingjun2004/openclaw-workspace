# Eclipse Mosquitto (MQTT Broker - 老牌轻量)

## 基础信息
- **项目名**: eclipse/mosquitto
- **URL**: https://github.com/eclipse/mosquitto
- **作者/团队**: Eclipse Foundation
- **License**: EPL-2.0 / EDL-1.0 ✅
- **语言**: C
- **平台**: Linux/macOS/Windows
- **Star 数**: ~9,000+
- **维护状态**: 活跃

## 功能介绍
- **轻量级 MQTT Broker (老牌)**
- 单机即可运行
- 适合小到中型 IoT 部署
- MQTT 5.0/3.1.1

## 适合嵌入到蓝领金服 app 的场景
- S 类 - 小型工地 IoT 消息
- Q 类 - 楼宇 IoT
- 蓝领金服 app 工地端小型 IoT 部署 (vs EMQX 适合大规模)

- 大量设备需用 EMQX 或 HiveMQ
