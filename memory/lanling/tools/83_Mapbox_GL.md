- **项目名**: maplibre/maplibre-gl-js
- **URL**: https://github.com/maplibre/maplibre-gl-js
- **作者/团队**: MapLibre (community fork of Mapbox GL)

- **Star 数**: ~10,000+
- **维护状态**: 极其活跃

## 功能介绍
- **开源矢量地图渲染 (Mapbox 开源分叉)**
- 蓝领金服 app 工地端"工地地图"模块
- 工地车辆轨迹 + 工人定位
- 集成 BIM 3D Tiles

## 适合嵌入到蓝领金服 app 的场景
- S 类 - 智慧工地 (车辆/人员定位)
- A 类 - 现场管理 (地图看板)
- 国产瓦片源 (天地图/高德) 适配
