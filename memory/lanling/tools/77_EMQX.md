# EMQX (国产 MQTT Broker - 蓝领金服 app 优选)

## 基础信息
- **项目名**: emqx/emqx
- **URL**: https://github.com/emqx/emqx
- **作者/团队**: EMQ Technologies (中国杭州,杭州映云科技)
- **License**: Apache-2.0 ✅ (开源版)
- **语言**: Erlang + Elixir
- **平台**: Docker / Linux / Kubernetes
- **Star 数**: ~15,000+
- **维护状态**: 极其活跃 (国产 MQTT Broker 头部)

## 功能介绍
- **大规模 IoT MQTT Broker (全球领先)**
- 单集群支持 1 亿+ MQTT 客户端连接
- MQTT 5.0 + 3.1.1
- 规则引擎 + 数据桥接 (Kafka/PostgreSQL/MongoDB/...)
- 物联网/工业互联网 通用
- **蓝领金服 app 工地端 IoT 设备连接首选**

## 技术栈
- Erlang OTP (电信级)
- 高并发分布式
- 多语言客户端 SDK
