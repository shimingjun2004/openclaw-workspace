# Apache Superset (BI 平台)

## 基础信息
- **项目名**: apache/superset
- **URL**: https://github.com/apache/superset
- **作者/团队**: Apache Software Foundation
- **License**: Apache-2.0 ✅
- **语言**: Python + TypeScript (React)
- **平台**: Web + Docker
- **Star 数**: ~65,000+
- **维护状态**: 极其活跃 (顶级 Apache 项目)

## 功能介绍
- **现代企业级 BI 平台 (Airbnb 起源)**
- 50+ 数据源
- SQL 实验室 (Notebook 式查询)
- 40+ 可视化图表
- 仪表板 + 告警 + 权限

## 技术栈
- Python + Flask
- React + TypeScript
- 多数据库
- Docker / K8s
