- **Star 数**: ~80,000+ (top 50 GitHub overall)

## 功能介绍
- **开源智能家居/楼宇自控 (BA) 平台 - 业界事实标准**
- 3000+ 设备/品牌 集成 (Zigbee/Z-Wave/WiFi/MQTT/Modbus/BACnet)
- 自动化引擎 (YAML/Python)
- 能源管理 + 仪表板
- iOS/Android App
- 家庭/楼宇/酒店/商场通用

## 技术栈
- Python 3.12 + asyncio
- Docker / HAOS 操作系统
- 集成生态 3000+
