## 功能介绍
- **完整的安全隐患闭环管理平台** (现场拍照上报 → 管理员审核分派 → 责任人限期整改 → 验收人闭环归档)
- **多租户设计** - 不同机构数据隔离
- **可视化智能看板** - ECharts 隐患分类、整改完成率、趋势
- **跨平台** - 微信小程序 + iOS/Android/鸿蒙 原生端
- **现代化架构** - Vite 8 + FastAPI 异步

## 技术栈
- 前端 (Admin): React + Vite 8
- 前端 (小程序): uni-app x + UTS + Vue 3
- 后端: Python + FastAPI
- 部署: 前后端分离
