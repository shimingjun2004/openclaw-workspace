# OpenCV (计算机视觉基础库)

## 基础信息
- **项目名**: opencv/opencv
- **URL**: https://github.com/opencv/opencv
- **作者/团队**: OpenCV 团队 (国际)
- **License**: Apache-2.0 ✅
- **语言**: C++ + Python/Java 绑定
- **平台**: 跨平台
- **Star 数**: ~80,000+
- **维护状态**: 极其活跃

## 功能介绍
- **开源计算机视觉事实标准库**
- 图像/视频处理
- 目标检测/跟踪
- 深度学习模型集成 (ONNX/TensorFlow/PyTorch)
- 蓝领金服 app 工地 AI 视觉底层依赖
