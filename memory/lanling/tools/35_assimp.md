## 功能介绍
- **3D 资源导入库 (Open Asset Import Library)**
- 支持 40+ 3D 格式
- 包括 IFC / glTF / FBX / STL / OBJ / COLLADA
- 统一数据模型
- C++/Python/C#/JS 绑定

## 技术栈
- C++ 核心
- 多语言绑定
- 部署: Conan / vcpkg / npm
