## 功能介绍

- IFC2x3 / IFC4 读写
- COBie 数据导出
- 与 Revit 集成
- 适合 .NET 生态

## 适合嵌入到蓝领金服 app 的场景
- C 类 - BIM 数据提取 (.NET 端)
- I 类 - 工程文档
- 偏 .NET 生态,蓝领金服如果是 Java 则不直接嵌入
