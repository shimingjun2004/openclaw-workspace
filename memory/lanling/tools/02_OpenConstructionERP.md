## 功能介绍
- 71 个模块,完整覆盖建筑 ERP
- BOQ (工程量清单) + PDF/CAD/BIM 算量
- AI 成本匹配 + 21 种语言
- 42 个地区材价目录
- MasterFormat / IFC / PDF / Revit 集成
- 一键 pip install 即可启动

## 技术栈
- 前端: TypeScript
- 后端: Python + FastAPI
- 数据库: PostgreSQL / SQLite
- 部署: Docker / pip / 单进程

## 适合嵌入到蓝领金服 app 的场景
- A 类 - 现场管理 ERP 全套 (进度/工人/文档)
- B 类 - 算量/计价/材价 (核心)
- I 类 - 工程资料归档
- J 类 - 工人管理/工资/合同
