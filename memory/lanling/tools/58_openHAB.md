# openHAB (open Home Automation Bus)

## 基础信息
- **项目名**: openhab/openhab-core
- **URL**: https://github.com/openhab/openhab-core
- **作者/团队**: openHAB Foundation (德国/欧洲基金会)
- **License**: EPL-2.0 ✅
- **语言**: Java
- **平台**: Linux/Windows/macOS/Docker
- **Star 数**: ~5,000+
- **维护状态**: 极其活跃 (基金会 + 全球贡献者)

## 功能介绍
- **开源楼宇自控/智能家居平台 (Java 生态)**
- 200+ 设备绑定 (Bindings)
- 规则引擎 + 自动化
- HABot 聊天机器人
- 适合商业楼宇/工业

## 技术栈
- Java + OSGi (Eclipse)
- Eclipse Smart Home 框架
- Docker / Linux 部署
