- **最近更新**: 2026 (活跃)

## 功能介绍
- **建筑能耗/采光/通风/热舒适模拟生态**
- Ladybug: 焓湿图、日照、采光分析
- Honeybee: EnergyPlus/IES/Radiance 接口
- Dragonfly: 城市/区域尺度
- 业内设计师人手一套

## 技术栈
- Python + IronPython (Rhino/Grasshopper 集成)
- 可视化: Plotly / Matplotlib / Three.js
