# OpenProject

## 基础信息
- **项目名**: opf/openproject
- **URL**: https://github.com/opf/openproject
- **作者/团队**: OpenProject Foundation (德国非营利组织)
- **License**: GPL-3.0 (社区版)
- **语言**: Ruby (后端) + TypeScript/Angular (前端)
- **平台**: Web (SaaS / 自托管)
- **最近更新**: 2026-06-06 (活跃)
- **Star 数**: ~15,200
- **Fork 数**: ~2,800
- **首次提交**: 2011
- **Issues 活跃度**: 高 (1k+ open)
- **维护状态**: 活跃 (基金会持续维护)

## 功能介绍
- 项目计划 + 甘特图 + 关键路径 (BCF 支持)
- 任务/工时/工单管理
- 工作流、看板、Scrum
- 文档管理与版本控制
- BIM 协作 (支持 IFC/BCF)
- 团队和权限管理
- 多语言 (含中文)

## 技术栈
- 前端: Angular + TypeScript
- 后端: Ruby on Rails
- 数据库: PostgreSQL
- 部署: Docker / Kubernetes / Linux

## 风险
- License: **GPL-3.0** - 衍生作品需开源,嵌入需谨慎 (建议作为外部服务通过 API 集成,不开源衍生)
- 重型系统,不适合做"小工具"模块,适合做"独立子产品"
- 部署依赖多,运维成本高
