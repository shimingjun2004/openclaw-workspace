## 功能介绍
- **世界首个完全基于浏览器的 DXF/DWG 查看器和编辑器**
- 无需后端,纯前端运行
- 支持 DWG 2000-2024 全版本
- 支持 R13-R15 DXF
- 在线编辑/标注/打印
- 移动端可用
- 三维预览

## 技术栈
- 前端: TypeScript + Three.js
- WebGL + WebAssembly
- 部署: CDN / npm / 单文件
