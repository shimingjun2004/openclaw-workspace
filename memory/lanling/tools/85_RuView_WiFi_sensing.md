## 功能介绍
- **π RuView: 把 WiFi 信号变成实时空间感知**
- 不需要摄像头 → 隐私友好的工人存在/姿势检测
- 心率/呼吸/跌倒检测
- ESP32 固件 + REST API
- 与 Home Assistant 集成

## 适合嵌入到蓝领金服 app 的场景
- W 类 - **工地 AI 视觉 (无摄像头隐私) ** ⭐⭐⭐⭐⭐
- A 类 - 现场管理 (工人存在/位置)
- 工地宿舍/卫生间/特殊区域无需摄像头的人存在检测
- 蓝领金服 app "智慧宿舍"模块
