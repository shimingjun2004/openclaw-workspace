# Node-RED (流式 IoT 编程)

## 基础信息
- **项目名**: node-red/node-red
- **URL**: https://github.com/node-red/node-red
- **作者/团队**: OpenJS Foundation (IBM 起源)
- **License**: Apache-2.0 ✅
- **语言**: JavaScript + Node.js
- **平台**: 跨平台 (Node.js 即可)
- **Star 数**: ~20,000+
- **维护状态**: 极其活跃

## 功能介绍
- **可视化流式编程工具 (低代码)**
- IoT 设备/协议 集成
- MQTT/HTTP/MODBUS/OPC-UA
- 楼宇/工厂/工地 自动化逻辑编排
- 蓝领金服 app 工地端"自动化场景"模块借鉴

## 技术栈
- Node.js + Express
- 浏览器流编辑器
- 4000+ 节点 (扩展)

## 演示
- 官网: https://nodered.org
- 在线: http://nodered.org/docs/getting-started/

- Q 类 - 智能建筑 自动化编排
