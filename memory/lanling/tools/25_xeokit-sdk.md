## 功能介绍
- **3D BIM/IFC 查看器 SDK**
- 纯 WebGL,无第三方渲染引擎
- 大模型性能 (double precision)
- 点云查看 (LAS/LAZ)
- 3D Tiles
- 真实世界坐标
- 适合大型工程模型

## 技术栈
- TypeScript + WebGL 2.0
- 无 Three.js 依赖
- 部署: npm / CDN
