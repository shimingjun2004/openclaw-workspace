# ThingsBoard (IoT 平台 - 设备管理/数据可视化)

## 基础信息
- **项目名**: thingsboard/thingsboard
- **URL**: https://github.com/thingsboard/thingsboard
- **作者/团队**: Thingsboard Inc (美国/乌克兰)
- **License**: Apache-2.0 ✅ (社区版) / 商业版额外功能
- **语言**: Java + Angular
- **平台**: Web + Docker + Kubernetes
- **Star 数**: ~18,000+
- **维护状态**: 极其活跃

## 功能介绍
- **开源 IoT 平台 - 设备管理 + 数据采集 + 规则引擎 + 可视化**
- 楼宇/工厂/工地 通用
- 50+ 设备协议 (MQTT/CoAP/LWM2M/Modbus/BACnet)
- 实时仪表板 + 历史数据
- 规则链 (可视化)
- 边缘部署 (ThingsBoard Edge)

## 技术栈
- Java + Spring + Akka
- Angular 前端
- PostgreSQL + Cassandra/HSQLDB
- Docker / K8s
