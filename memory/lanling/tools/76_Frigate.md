## 功能介绍
- **NVR (网络视频录像机) + 实时本地目标检测**
- IP 摄像头 RTSP 输入
- 集成 YOLO + Google Coral (TPU) 边缘推理
- 与 Home Assistant 深度集成
- MQTT 事件推送
- **完美适合蓝领金服 app 工地端 "AI 监控"模块**

## 技术栈

- Docker
